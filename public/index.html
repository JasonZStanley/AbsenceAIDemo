Overview

This service will parse an audio clip and convert it into text. It will then attempt to extract the childs name, along with the reason and duration for absence

The following Postman collection can be used to interact with the service: Postman Collection

Current limitations

This is a basic PoC.

  • A fairly basic transcription model is used as a decent GPU is required to run larger models in a timely fashion. A better model will have better accuracy
  • What is being pulled out of the text can be improved and extended.
  • Generally speaking, the code does enough to function but lacks error handling, ability to scale, validation etc etc
